# Agent Skills

A collection of reusable agent skills for AI coding assistants.

## Skills

### [agent-init](./skills/agent-init/SKILL.md)
Generate AGENTS.md documentation for codebases to guide Code Agent behavior. Use when initializing AI coding assistant configurations, standardizing agent behavior, or creating agent guidance docs for new projects.

### [prompt-writer](./skills/prompt-writer/SKILL.md)
Provides systematic LLM prompt writing and optimization guidance. Use when users need to create new prompts, optimize existing prompts, learn prompt engineering techniques, or improve prompt quality.

### [prompt-writer-zh](./skills/prompt-writer-zh/SKILL.md)
Chinese version of the prompt-writer skill.
